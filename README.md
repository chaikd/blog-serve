# 介绍
本项目是为个人博客项目提供的API服务端项目，前端项目地址[请点击](https://github.com/chaikd/blog.git)。
# 启动项目
	  # clone the project
	  git clone https://github.com/chaikd/blog-serve.git

	  # enter the project directory
	  cd blog-serve

	  # install dependency
	  npm install

	  # develop
	  npm run start
